# ⏱️ PyQt5-Stopwatch

A simple desktop stopwatch application built using Python and PyQt5. The program measures elapsed time with millisecond precision and allows users to start, stop, and reset the timer through a graphical interface.

## 🛠️ Features

- Start / Stop Timer: Begin and pause the stopwatch with interactive buttons.

- Reset Function: Reset the timer back to zero instantly.

- Real-Time Display: Updates every 10 milliseconds using QTimer.

- Clean GUI Layout: Built with PyQt5 using labels, buttons, and layouts.

## ⚙️ How It Works

The stopwatch tracks elapsed time using QTime and updates the display through a QTimer that triggers every 10 milliseconds. The current elapsed time is formatted and displayed in HH:MM:SS.ms format.
